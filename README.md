# Indian Air Quality Analysis (AQI)

## Project Overview
This project analyzes Air Quality Index (AQI) data from different cities in India using Python and Power BI. The goal is to identify pollution trends, compare cities, and understand the impact of pollutants on air quality.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Power BI

## Dataset Information
- Dataset: Indian Air Quality Index (AQI)
- Records: 29,531
- Columns: 16

## Data Cleaning
- Missing value detection
- AQI median replacement
- Date formatting
- Duplicate removal
- Feature engineering

## Analysis Performed
- AQI Trend Analysis
- Top Polluted Cities Analysis
- AQI Distribution Analysis
- Health Risk Classification
- PM2.5 vs AQI Analysis

## Key Findings
- Delhi recorded very high AQI levels.
- PM2.5 strongly influenced AQI values.
- Urban cities showed severe pollution levels.
- Seasonal AQI fluctuations were observed.

## Dashboard
The project includes an interactive Power BI dashboard for visualizing air quality trends across Indian cities.
